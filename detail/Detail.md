# プロフィール詳細

## 今後作りたいもの（アイデア）

| アイデア                                                      | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| ------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 遊戯王の最終展開盤面                                          | 遊戯王のテーマやカードの検索から展開ルートの最終盤面を検索できるアプリ。ブログとか Youtube で学ぶことが多かったので、簡単に盤面が視覚的にわかるアプリを作成したい。[GitHub で開発中](https://github.com/ryomeblog/yugioh-final-field)                                                                                                                                                                                                                        |
| 分散型 Twitter ライクアプリ                                   | ツイートは全て DB ではなくキャッシュに持つ。ユーザがログインしたときに最新ツイートがキャッシュに保持される。キャッシュ内のツイートがホームで表示される。サーバ側のキャッシュは 1 日くらいで、ツイートの保持期間もそれくらい。いいねとか返信したツイートは各自の端末に保持されている。自分のツイートの返信が 1 日経ったら消えるので気付けないのとツイート改ざんされる対策については今後考えたい。ツイート保持サーバを基準さえクリアすれば誰でも立てれるとか。 |
| ページ内の繰り返し要素を json, csv に変換する Chrome 拡張機能 | Chrome 拡張機能でボタンを押すと、正規表現を入力して繰り返し要素の中身を json でダウンロードできる。設定で csv にもできる。                                                                                                                                                                                                                                                                                                                                   |
| スマホに特化したフローチャート作成アプリ                      | スマホでフローチャートを書きたい、見たい。json か CSV でエクスポート出来るようにしたい。作ったフローチャートからインプットとアウトプットだけ決めて Swagger に直すとかしたい。中処理 AI に欠かせられたら尚良い。                                                                                                                                                                                                                                              |
| 目覚ましアプリ                                                | アラームをグループ化できる。1 つのアラームを停止するとそのグループのアラームはその日はならなくなる。（5 分間隔でアラームセットしたけど起きたから全て止めたい時に便利）<br/>スマートスピーカーと連動して、「〜グループに ○ 時のアラーム追加して」で追加できる。                                                                                                                                                                                               |
