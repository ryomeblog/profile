## 経歴

### 2024 年

**■ プロジェクト一覧**

- 観光向け予約管理SaaSの自社開発

**■ 技術キーワード**

Docker, ECS, CloudFormation, CDK, IaC, Cognito, CodeCommit, CodeBuild, CodeDeploy, CodePipeline, CI/CD, WAF, Lambda, S3, CloudFront, DynamoDB, RDS, React, Next.js, TypeScript, Playwright, Java, Spring, Gradle, Open API, Swagger, スクラム開発

**■ 担当業務**

- E2E テスト自動化によるテスト工数の削減
- セキュリティの見直し
- Next.js における SSR と SSG を考慮した開発
- OpenAPI を利用したバックエンド IF の自動生成
- 多言語化対応（i18nを利用した言語ツリー設計）
- リーダー（テックリード）として AI 開発導入のアプローチやスクラム開発の推進、コードレビュー、タスク管理、出荷審査等

**■ 発揮したバリュー**

- プロジェクト立ち上げ時の整備

プロジェクト立ち上げ時に整備されていなかったブランチのバージョン管理方法やフロントエンド開発の標準などを再精査してプロジェクトの円滑な進行に貢献。
また、GitHub Copilot や Cline などを使用した AI 開発の導入を行い、開発工数の削減にも寄与。

- E2E テストの自動化によるテスト工数の削減

Playwright による E2E テストの自動化を CI/CD パイプラインに組み込むことでテスト工数の大幅な削減に成功。
E2E テストでは、MFA 認証を突破するユースケースなどの難しい要件にも対応。

- セキュリティの見直し

S3 バケットのパブリック設定（CloudFront 経由のアクセスに変更）やセキュリティグループのなぜか空いているポート、使われない NAT Gateway などセキュリティ面に問題のある個所を修正することでプロジェクトのセキュリティ向上に貢献。

**■ 技術キーワード**

OpenShift（Kubernetes）, Docker, CloudFormation, CDK, IaC, Cognito, CodeCommit, CodeBuild, CodeDeploy, CodePipeline, CI/CD, WAF, Amplify, Figma, AppSync, GraphQL, Lambda, S3, CloudFront, DynamoDB, RDS, Kubernetes, ROSA, React, Jest, Node.js, Python, Pandas, Git

**■ 担当業務**

- インフラ構成検討、データ設計
  - Amplify を利用した Figma から React コード生成および開発
  - GraphQL スキーマ定義
  - 試験自動化導入
- Git 管理、ROSA（OpenShift）を利用した開発基盤実装、ネットワーク検討・設計
- Pandas の高速化に挑戦

**■ 発揮したバリュー**

- OpenShift（Kubernetes）の特殊要件対応

OpenShift 特有の特殊要件（コンテナから Lambda を経由してメール送信や印刷、対象コンテナ全てにガベージコレクションをかけるなど）に対応。

- AppSync の Subscription によるリアルタイム性の実現

WebSocket を利用したチャットアプリの既読機能や即時チャット反映などリアルタイム性が求められるユースケースに対応。
CDK を使用してインフラをコード管理することで、再現性のあるインフラ構成と手動構築コストを削減し、プロジェクトの生産性に大きく貢献。

- Amplify 利用による開発効率化

技術選定の段階で Amplify × Figma を選択することで、通常では３ヶ月程度に見積もられていた開発を１か月程度に短縮することに成功。
また、Jest による試験のコード化や CI/CD パイプライン内の試験自動化を組み込み、プロジェクトの生産性向上と工数削減に大きく貢献。

- SaaS, Office 連携対応

Webhook を利用した SaaS 間の連携 や AWS, SaaS のエラー通知をメールまたは Teams に送るなど SaaS を利用した開発を経験。

### 2021 年

**■ プロジェクト一覧**

- 光回線事業者向け契約管理システム開発
- 某通信建設会社向け販売管理 WEB システム改修

**■ 技術キーワード**

Java, Servlet, Spring, Angular, Seasar2, Jenkins, MyBatis, JUnit, Oracle, SQL Server, Subversion

**■ 担当業務**

- パラレル処理を用いたメール送信バッチ開発、SPA 構成の画面改修、外部 API と連携するバッチ改修
- Jenkins を利用した CI/CD パイプライン構築
- SQL インジェクション発生コード調査および改修
- JUnit による単体試験の自動化
- 新規参入者向けの環境整備

**■ 発揮したバリュー**

- モックサーバ構築の簡易化による品質向上

認証と決済処理に関して、バックエンド起動時にモックサーバを自動的に起動するツールを作成することで単体レベルのテストを可能にし、プロジェクトの品質向上に貢献。

- レガシーシステムの改修

誰も改修することができない 2000 年代初期の Java プロジェクトで発生した SQL インジェクションに対し、ソリューション（サニタイジングとバリデーションチェック、デッドロックの解消、JUnit によるテスト自動化等）を提案・実施し、技術的負債の解消に大きく貢献。
