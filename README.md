## 経歴

### 2024年

**■ プロジェクト一覧**

- 観光向けSaaSの自社開発

**■ 技術キーワード**

Docker, ECS, CloudFormation, CDK, IaC, Cognito, CodeCommit, CodeBuild, CodeDeploy, CodePipeline, CI/CD, WAF, Lambda, S3, CloudFront, DynamoDB, RDS, React, Next.js, TypeScript, Playwright, Java, Spring, Gradle, Open API, Swagger, スクラム開発

**■ 担当業務**

- E2Eテスト自動化によるテスト工数の削減
- セキュリティの見直し
- Next.jsにおけるSSRとSSGを考慮した開発
- OpenAPIを利用したバックエンドIFの自動生成
- 多言語化対応
- リーダーとしてのテックリードやコードレビュー、タスク管理、出荷審査等

**■ 発揮したバリュー**

- ブランチ戦略の提案

ブランチのバージョン管理をセマンティックバージョニングにするなどプロジェクト立ち上げ時に整備されていない箇所を洗い出し、再精査してプロジェクトを円滑な進行に貢献。

- E2Eテストの自動化によるテスト工数の削減

PlaywrightによるE2Eテストの自動化をパイプラインに組み込むことでテスト工数の大幅な削減に成功。
E2Eテストでは、MFA認証を突破するユースケースなどの難しい要件にも対応。

- セキュリティの見直し

S3バケットのパブリック設定（CloudFront経由のアクセスに変更）やセキュリティグループのなぜか空いているポートなどセキュリティ面に問題のある個所を修正することでプロジェクトのセキュリティ向上に貢献。
